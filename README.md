# Decentralized Crowdfunding Contract

This project implements a decentralized crowdfunding platform using smart contracts. It allows users to create campaigns, contribute funds, and manage campaigns transparently on the blockchain.

## Features

- Create and manage crowdfunding campaigns.
- Contribute funds securely.
- Transparent and immutable records.

## Technologies Used

- Solidity
- Ethereum
- Hardhat
- JavaScript/TypeScript

## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/johnomo742/Decentralized-Crowdfunding-Contract.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Compile the smart contracts:
    ```bash
    npx hardhat compile
    ```
4. Run tests:
    ```bash
    npx hardhat test
    ```
